# Getting started

## Requirements

- [Nodejs](https://nodejs.org/en/download/) version 16.4 or higher
    - When installing Node.js, you are recommended to check all checkboxes related to dependencies.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development

$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
### Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
